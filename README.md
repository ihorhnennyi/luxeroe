<h1 align="center">🛍️ Ikra Landing — Premium Caviar Store</h1>

<p align="center">
  <b>SEO-optimized landing page with product catalog and cart functionality</b><br/>
  <sub>Built with Next.js | Plain CSS | Zustand | SEO Meta | Simple Order Form</sub>
</p>

---

## 🚀 Features

- 🔍 SEO-friendly with full meta and Open Graph support
- 🛒 Product cart with add/remove functionality
- 📦 Simple order form submission (no payment gateway)
- 📱 Responsive layout for mobile and desktop
- ⚙️ Server-side rendering for fast loading and SEO
- 📸 Product gallery with high-quality images

---

## 🧠 Technologies

| Technology      | Description                           |
| --------------- | ------------------------------------- |
| Next.js         | React framework with SSR/SSG          |
| Zustand         | Lightweight global state for cart     |
| CSS Modules     | Scoped CSS for styling                |
| React Hook Form | Form management and validation        |
| SEO Meta Tags   | Custom meta tags + OpenGraph + schema |

---

## 🛠️ Installation

```bash
git clone https://github.com/ihorhnennyi/ikra-landing.git
cd ikra-landing

# Install dependencies
npm install

# Start the development server
npm run dev
```
